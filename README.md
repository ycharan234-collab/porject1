# 💊 Medicine Reminder Web Application

A simple and user-friendly Medicine Reminder web application that helps users schedule and manage their daily medication reminders. The application allows users to register, log in, add medicines with dosage and timing, edit or delete reminders, and receive notification alerts when it's time to take their medicine.


## 🚀 Features
- 👤 User Registration & Login
- 🔓 Guest Mode
- 💊 Add Medicine Reminders
- ✏️ Edit Existing Reminders
- 🗑️ Delete Reminders
- 📅 Select Days of the Week
- ⏰ Set Reminder Time (AM/PM)
- 🔔 Browser Notification Alerts
- 🔊 Audio Reminder Alerts
- 📊 Dashboard to View All Medicines
- 💾 Local Storage Support (No Database Required)
- 📱 Responsive User Interface

## 📖 How It Works
- Create a new account or continue as a Guest.

- Add medicine details such as:

  - Medicine Name

  - Dose

  - Reminder Time

  - Days of the Week

- Save the reminder.

- View, edit, or delete reminders from the dashboard.

- Receive browser and sound notifications when a reminder is due.
